üì± ISBN Scanning - Three Methods

üì∑ Photo Capture

  • Use your device camera to take a photo of the ISBN barcode
  • Advanced interactive image cropping tool to focus on the barcode area
  • Drag-and-drop crop handles with edge and corner resizing
  • OCR (Optical Character Recognition) powered by Tesseract.js
  • Automatically extracts ISBN from the photo using text recognition
  • Edit detected ISBN if recognition needs correction
  • Supports both ISBN-10 and ISBN-13 format detection

üìä Real-time Barcode Scanning

  • Live barcode scanning using device camera with Quagga2 library
  • Instant ISBN detection with visual feedback
  • Supports multiple barcode formats: EAN-13, EAN-8, UPC-A, UPC-E, and Codabar
  • Fast and accurate scanning technology with real-time processing
  • Automatic camera stream management

‚úèÔ∏è Manual ISBN Input

  • Type or paste ISBN directly
  • Accepts ISBN with or without hyphens
  • Perfect for when scanning isn't available

üìö Automatic Book Information Retrieval

  • Fetches book details from Google Books API using ISBN
  • Automatically populates title and author information
  • Fallback to Open Library API if Google Books doesn't have results
  • Automatic ISBN-13 to ISBN-10 conversion for additional lookup attempts
  • Validates ISBN format (10 or 13 digits without hyphens)
  • 8-second timeout per API request to prevent hanging
